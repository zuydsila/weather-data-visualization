# Weather Data Visualization

## Overview

This project provides an interactive dashboard to explore weather data using dynamic visualizations.

## Features

* Interactive charts
* Data filtering
* Dynamic updates
* Tooltips showing detailed values

## Technologies

* Python
* Flask
* JavaScript
* D3.js
* SQLite

## Project Structure

weather-data-visualization/
│
├── static/
├── templates/
├── app.py
└── README.md

## How to Run

Clone the repository:

git clone https://github.com/yourusername/weather-data-visualization.git

Install dependencies:

pip install flask

Run the server:

python app.py

Open the browser at:

http://localhost:5000

* Building data visualizations
